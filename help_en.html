Handling

The Map extent can be moved by a finger wipe. By spreading two fingers or a double tapping you can zoom in. If you tighten two fingers, you can zoom out.

By tapping on a symbol you get an information about it.

The user has five buttons with different functions.

opens the topics and layers dialog
search for addresses or objects
shows the current location of the user
measure, E-Mail, Print PDF
language, Impressum and other options

Topics and Layers

Here the user can choose between three tabs (Topics, Layers, Order)

In Topics you can open a map with a specific theme.

In Layers you can see all layers which belong to the current topic. Here you can turn a layer on and off.

In Order you can see all layers which are turned on at the moment. The user can change the the order of these layers by click or tap and move it up and down. The layer which is top in the list, is also top in the map view. That means it will hide the other layers underneath.

The activated layer (in dark grey) can be displayed with transparency, by moving the slider or typing the percentage. The higher the value, the more transparent is the layer.

Search

Here the user can type in every search term. The search works independent of upper and lower case and recognizes even parts of a search term.

The results are shown in the list underneath. There is also shown the name of the layer and the count of results in this layer. If you click or tap on a single result the object is centered in the map, marked with a red circle. If this object is part of a layer, which was not yet activated, it gets activated automatically.

Note: The user can only find objects, which are part of the actual map topic.

In Advanced Search the user can limit your search results.

With the filter you can determine for which specific layers results should be displayed.

With the destrict based search you can limit the search results to one district.

With a Search radius you can define search results for one specific radius. As long as the search radius is activated, the user can position the circle by click/tap in the map. By moving the slider the user can change the perimeter of the circle.

Location

Click or tap the button and the current location of the user is shown in the map.

Tools

With Share map you can send the current map view in an E-Mail with an URL. The center of the map is marked. with a pin, which the user also can fade out. A click on send opens an E-Mail and inserts the URL.

With Get coordinates the user can click in the map to get the coordinates of this position.

With Measure the user can calculate a distance or an area. A double tap finishes the measure and shows the result.

with Print PDF you can export an area in a predefined layout as PDF. At first the user has to choose the format and the scale. Print creates the document and shows it in the browser. There you can save it or print it out.

Settings

The user can switch between German and English. This relates to the buttons, the menu and the support texts.

Map following means the current map view follows the users movement. Map rotation means the map rotates by following the viewing direction of the user. If Map rotation is turned off, the map is always oriented in a northward direction.

The user can decide if a scale bar is displayed or not. Even two buttons for Zoom In / Zoom Out can be displayed, when the user isn't comfortable with zooming with a finger wipe.
